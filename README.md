## Abstract
Participants will take a deep dive into architectural concepts, cloud fundamentals, Azure services, and DevOps exposure all while building serverless computing apps using the latest technology from Microsoft - Azure Functions. Participants will leverage the power of the cloud to deploy and run their very own Azure Functions. Participants will leave the workshop with the ability to use Domain Driven Design concepts paired with serverless programming to tackle many complex business problems facing enterprises today.
